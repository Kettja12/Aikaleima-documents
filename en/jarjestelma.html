System Configuration

The system consists of the following parts. Traffic from the wildcard DNS name *.aikaleima.fi is directed to an Nginx proxy server installed on a Linux virtual server. The proxy server directs traffic to different Blazor Server instances, which are installed on the same Linux virtual server. Each customer has their own Blazor Server instance and their own SQLite database. The test environment has its own Blazor Server instance and its own SQLite database.
